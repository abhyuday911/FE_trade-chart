## Features

- Realtime updates with mocked crypto data
- Interactive charts using Chart.js
- Tabular view of coin prices and metrics
- Loading skeletons for better UX
- Data fetching, caching, and refetching with TanStack Query

## Tech Stack

- React
- @tanstack/react-query
- Chart.js
- Mocked data source
